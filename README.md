# spring-boot-zuul-demo

Example project for using a Zuul reverse proxy with Spring Boot.

## Getting Started

### Prerequisites

* Java SE 8 / Java 1.8
* Maven 3

### Installing

mvn clean install

## Running the tests

mvn clean install

## Running the app

mvn spring-boot:run

## Built With

* [Spring Boot](https://projects.spring.io/spring-boot/)
* [Maven](https://maven.apache.org/)
* [Zuul](https://github.com/Netflix/zuul)
